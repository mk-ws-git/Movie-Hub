Movie Hub is a Python-based command-line application that lets users manage a personal movie database and generate a static website to display the collection.

Features:

- Add, update, delete, and list movies 
- Search and filter movies by rating and year

- View statistics and generate rating histograms
- Generate a static website from an HTML template

Technologies:

- Python 3 
- SQLite (via SQLAlchemy)
- Matplotlib 
- HTML & CSS

Installation:
1. Clone the repository:
    git clone
2. Navigate to the project directory:
    cd Movie-Hub
3. Install the required dependencies:
    pip install -r requirements.txt

Usage:
To generate the static website, run the following command in your terminal:
python3 movies.py

View the Website:
1. Run the app and select Generate website from the menu
2. Open _static/index.html in a browser
